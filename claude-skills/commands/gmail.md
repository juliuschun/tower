# Gmail

사용자의 Gmail 작업을 수행합니다.

## 사용법

- `/gmail` — 최근 받은편지함 (5건)
- `/gmail inbox` — 받은편지함 조회
- `/gmail search 키워드` — 메일 검색
- `/gmail read 제목또는ID` — 메일 내용 읽기
- `/gmail send 수신자 제목` — 메일 작성/발송
- `/gmail draft 수신자 제목` — 임시저장 작성

## 실행

ToolSearch를 사용하여 `+gmail` 키워드로 MCP 도구를 로드한 후 실행하세요.

1. 먼저 `ToolSearch`로 `+gmail` 검색하여 필요한 도구 로드
2. 사용자 요청에 맞는 도구 실행
3. 결과를 한국어로 간결하게 요약

### 주의사항
- 메일 발송 전 반드시 사용자 확인 (수신자, 제목, 본문 미리보기)
- 메일 삭제는 각별히 주의
- 첨부파일 다운로드 경로 확인
