# Tower 도움말

Tower에 오신 것을 환영합니다. Tower는 팀을 위한 AI command center입니다. AI 대화, 팀 협업, 파일 관리, 태스크 자동화를 하나의 워크스페이스에서 처리합니다.

처음 사용하시는 분이든 특정 기능을 찾고 계신 분이든, 이 가이드가 도움이 될 것입니다.

---

## Tower란?

Tower는 팀 전체가 AI를 함께 사용할 수 있는 웹 플랫폼입니다. 개별 챗봇 대신, Tower가 제공하는 것은:

- **Sessions** -- AI와의 1:1 개인 대화
- **Channels** -- AI가 내장된 팀 채팅 공간
- **Files** -- 코드 에디터, 미리보기, 공유가 가능한 공유 파일 트리
- **Tasks** -- AI가 자율적으로 작업을 수행하는 칸반 보드
- **Visual Blocks** -- 차트, 다이어그램, 지도 등이 채팅에서 바로 렌더링

모든 것은 **Project** 단위로 정리되어, 관련 대화, 파일, 태스크가 함께 묶입니다.
