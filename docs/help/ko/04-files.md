# Files -- 파일 관리

> File Tree로 프로젝트 파일을 탐색하고, Context Panel에서 바로 보고 편집하세요. 공유와 다운로드도 한 번에.

---

## File Tree

Sidebar에서 **Files** 탭을 선택하면 파일 트리가 나타납니다.

### 기본 탐색

- **폴더 클릭** -- 확장/축소 토글
- **파일 클릭** -- Context Panel에서 열기
- 중첩된 폴더도 자유롭게 탐색할 수 있습니다

### 파일 아이콘 색상

파일 유형에 따라 아이콘 색상이 달라집니다.

| 색상 | 파일 유형 |
|------|----------|
| 파랑 | TypeScript (.ts, .tsx) |
| 노랑 | JavaScript (.js, .jsx) |
| 초록 | Python (.py) |
| 빨강 | HTML (.html) |
| 보라 | CSS (.css, .scss) |
| 회색 | 설정 파일 (.json, .yaml, .toml) |

---

## 숨김 파일

기본적으로 `.`으로 시작하는 숨김 파일(예: `.env`, `.gitignore`)은 표시되지 않습니다.

- File Tree 상단의 **숨김 파일 토글**을 클릭하면 표시/숨김 전환
- `node_modules`, `.git` 등 자주 숨기는 디렉토리도 이 토글로 제어됩니다

---

## 컨텍스트 메뉴 (우클릭)

파일이나 폴더를 우클릭하면 컨텍스트 메뉴가 나타납니다.

### 파일 메뉴

| 항목 | 설명 |
|------|------|
| **새 파일** | 현재 위치에 새 파일 생성 |
| **새 폴더** | 현재 위치에 새 폴더 생성 |
| **이름 변경** | 파일/폴더 이름 변경 (Enter로 확정, Escape로 취소) |
| **삭제** | 파일/폴더 삭제 (확인 후 진행) |
| **공유** | 공유 링크 생성 |
| **다운로드** | 파일 다운로드 |
| **경로 복사** | 파일 경로를 클립보드에 복사 |

### 폴더 메뉴

폴더 우클릭 시에는 위 항목에 더해 **폴더 안에 새 파일/폴더 생성** 옵션이 나타납니다.

---

## 파일 업로드

### 폴더에 드래그앤드롭

1. 탐색기(Finder)에서 파일을 선택합니다
2. File Tree의 원하는 폴더 위로 드래그합니다
3. 드롭 영역이 강조되면 놓습니다
4. 업로드 완료

여러 파일을 한 번에 드래그하여 업로드할 수 있습니다.

### 입력창에 드래그하여 첨부

Session이나 Channel의 입력창에 파일을 드래그하면, 메시지에 파일이 첨부됩니다. AI에게 파일을 분석하거나 수정을 요청할 때 가장 빠른 방법입니다.

File Tree에서 파일을 입력창으로 드래그하는 것도 가능합니다.

---

## Context Panel

파일을 클릭하면 화면 오른쪽 Context Panel에서 열립니다.

### 파일 유형별 동작

| 파일 유형 | Context Panel 동작 |
|----------|-------------------|
| **코드 파일** (.ts, .py, .js 등) | 구문 강조 코드 에디터 |
| **Markdown** (.md) | 렌더링된 프리뷰 (Mermaid 다이어그램 포함) |
| **PDF** (.pdf) | PDF 뷰어 |
| **이미지** (.png, .jpg, .gif 등) | 이미지 미리보기 |
| **비디오** (.mp4, .webm 등) | 비디오 플레이어 |
| **기타** | 텍스트로 표시 |

### 코드 편집

Context Panel의 코드 에디터에서 직접 파일을 수정하고 저장할 수 있습니다. 변경 사항은 즉시 디스크에 기록됩니다.

---

## 파일 공유

파일을 외부에 공유할 수 있습니다.

1. 파일 우클릭 -- **공유** 선택
2. 공유 링크가 생성됩니다
3. 링크를 복사하여 전달

공유 링크를 받은 사람은 별도 로그인 없이 파일에 접근할 수 있습니다.

---

## Pin 기능

자주 사용하는 파일을 Pin으로 고정하면 빠르게 접근할 수 있습니다.

### Pin 추가

- 파일 우클릭 -- **Pin** 선택
- 또는 Context Panel 상단의 Pin 아이콘 클릭

### Pin 목록 보기

Sidebar 하단의 **Pins** 버튼을 클릭하면 고정된 파일/세션 목록을 한 눈에 볼 수 있습니다.

---

## 팁

- **경로 복사 활용**: AI에게 "이 파일 수정해줘"라고 할 때, 경로를 복사해서 붙여넣으면 정확합니다.
- **Markdown 프리뷰**: 문서를 작성하면서 실시간으로 렌더링 결과를 확인하세요.
- **File Tree에서 입력창으로 드래그**: AI에게 파일 분석을 요청하는 가장 빠른 방법입니다.
- **숨김 파일 체크**: `.env`나 설정 파일을 찾을 수 없다면 숨김 파일 토글을 확인하세요.
- **Pin으로 정리**: 프로젝트에서 핵심 파일 5-6개를 Pin해두면 작업 효율이 크게 올라갑니다.
