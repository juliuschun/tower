# Settings — 설정

> 테마, 모델, 연동 서비스, 관리자 기능까지. Tower를 나에게 맞게 설정하세요.

---

## Settings 열기

Sidebar 하단의 **Settings** 아이콘을 클릭하면 설정 모달이 열립니다.

일반 사용자와 Admin 사용자의 설정 항목이 다릅니다.

---

## 테마 변경

Tower는 5가지 테마를 제공합니다.

| 테마 | 설명 |
|------|------|
| **Dark** | 어두운 배경, 눈의 피로를 줄여줍니다 (기본값) |
| **Light** | 밝은 배경, 환한 곳에서 작업할 때 |
| **Ocean** | 파란 계열 다크 테마 |
| **Forest** | 초록 계열 다크 테마 |
| **Aurora** | 보라/핑크 계열 다크 테마 |

Settings에서 원하는 테마를 선택하면 즉시 적용됩니다.

---

## 모델 선택

Session에서 사용할 AI 모델을 변경할 수 있습니다.

### 세션별 모델 변경

각 Session의 설정에서 개별적으로 모델을 선택할 수 있습니다. 작업 특성에 따라 모델을 바꾸세요.

- 복잡한 분석/코딩 → 고성능 모델
- 간단한 질문/번역 → 빠른 모델

### 기본 모델 설정 (Admin)

Admin은 시스템 전체의 기본 모델을 설정할 수 있습니다:

| 용도 | 설명 |
|------|------|
| **Session 기본 모델** | 새 Session 생성 시 기본으로 사용되는 모델 |
| **AI Reply 모델** | Channel에서 @ai 응답에 사용되는 모델 |
| **AI Task 모델** | @task 실행에 사용되는 모델 |

## OAuth 연동

외부 서비스와 Tower를 연결합니다.

### 카카오톡

카카오톡 계정으로 로그인하거나, 카카오톡 알림을 받을 수 있습니다.

1. Settings → **연동 서비스** 메뉴
2. 카카오톡 **연결** 클릭
3. 카카오 인증 페이지에서 로그인 및 권한 승인
4. 연동 완료

연동을 해제하려면 같은 메뉴에서 **연결 해제**를 클릭합니다.

## Skills Market

Tower의 기능을 확장하는 Skills를 설치할 수 있습니다.

### Skills란?

Skills는 AI에게 새로운 능력을 부여하는 확장 패키지입니다. 예를 들어:

- 특정 프레임워크에 대한 전문 지식
- 코드 스타일 가이드 적용
- 도메인별 전문 프롬프트

### 설치 방법

1. Settings → **Skills** 메뉴
2. 사용 가능한 Skills 목록 탐색
3. 원하는 Skill의 **설치** 버튼 클릭
4. 설치 완료 후 즉시 사용 가능

## Admin 전용 기능

Admin 계정은 시스템 관리 기능에 접근할 수 있습니다. Header의 **방패 아이콘**을 클릭하면 Admin Panel이 열립니다.

### 사용자 관리

| 기능 | 설명 |
|------|------|
| **사용자 목록** | 등록된 모든 사용자 확인 |
| **사용자 비활성화** | 특정 사용자의 접근을 차단 (soft-delete) |
| **폴더 제한** | 사용자별 접근 가능 폴더 설정 (allowed_path) |
| **역할 변경** | 사용자를 Admin으로 승격 또는 해제 |

### 그룹 관리

사용자를 그룹으로 묶어서 프로젝트/채널 권한을 일괄 관리합니다.

### 모델 설정

시스템 전체에서 사용 가능한 AI 모델을 관리합니다.

- 사용 가능 모델 목록 설정
- Session/AI Reply/AI Task 별 기본 모델 지정
- 모델별 사용 제한 설정

### System Prompt 관리

시스템 전체에 적용되는 System Prompt를 설정합니다. 모든 AI 대화에 공통으로 적용되는 지침을 지정할 수 있습니다.

### Skills 관리

Admin은 시스템에 설치된 Skills를 관리합니다.

- 설치된 Skills 목록 확인
- 특정 Skills 비활성화
- Skills 업데이트

## 팁

- **테마는 취향대로**: 긴 시간 작업한다면 Dark 계열이 눈에 편합니다.
- **모델 선택 전략**: 일상적인 질문은 빠른 모델, 중요한 코드 작업은 고성능 모델을 사용하세요.
- **Admin 권한 주의**: 사용자 비활성화는 즉시 적용됩니다. 실수로 자신을 비활성화하지 않도록 주의하세요.
