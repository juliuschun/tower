## 프롬프트 팁

### 맥락을 충분히 주세요

```
❌ "보고서 만들어줘"
✅ "3월 매출 데이터(docs/sales-march.xlsx)를 기반으로
   경영진 보고용 요약 보고서를 만들어줘.
   핵심 지표 3개와 전월 대비 증감을 포함해줘."
```

### 형식을 지정하세요

```
"표로 정리해줘"
"차트로 보여줘"
"3줄로 요약해줘"
"마크다운으로 작성해줘"
"발표자료 형태로 만들어줘"
```

### 단계별로 나눠서 요청하세요

복잡한 작업은 한 번에 다 시키기보다 나눠서:
1. "먼저 경쟁사 3곳 조사해줘"
2. "좋아, 이 내용으로 비교표 만들어줘"
3. "이걸 바탕으로 우리 전략 제안해줘"

## 시각화 활용

AI에게 데이터를 주면 자동으로 시각화합니다:

| 이렇게 말하면 | 이런 시각화가 나옵니다 |
|-------------|---------------------|
| "매출 추이를 보여줘" | 꺾은선 차트 (line) |
| "부서별 비율을 보여줘" | 파이 차트 (pie) |
| "장단점 비교해줘" | 비교 카드 (comparison) |
| "프로세스를 그려줘" | 플로차트 (mermaid) |
| "일정을 정리해줘" | 타임라인 (timeline) |
| "데이터를 표로 보여줘" | 데이터 테이블 (datatable) |

## 채널 AI 활용

### @ai — 팀 전체가 보는 빠른 답변
- 회의 중 빠른 팩트체크
- 공유 맥락에서 질문 (모두가 답변을 봄)
- 이전 @ai 대화를 기억함 → 후속 질문 가능

### @task — 시간이 걸리는 작업
- 긴 리서치, 보고서 작성
- 파일 생성/수정이 필요한 작업
- 칸반 보드에서 진행 상황 추적

### 팁: 언제 세션, 언제 채널?
| 상황 | 추천 |
|------|------|
| 개인 작업, 반복 실험 | **세션** |
| 팀이 함께 볼 결과물 | **채널 @ai** |
| 30분 이상 걸리는 작업 | **채널 @task** |
| 민감한 정보 포함 작업 | **세션** (비공개) |

## 파일 & 프로젝트 팁

- **프로젝트 안에서 작업**: 프로젝트 세션을 열면 AI가 해당 프로젝트 파일을 자동 참조
- **파일 이름 구체적으로**: `보고서.md` 대신 `2026-04-매출분석-보고서.md`
- **decisions/ 활용**: 중요한 결정은 "이거 decisions에 기록해줘"라고 요청

## 스킬 조합 예시

### 시장 조사 → 보고서 → 발표자료

```
1. "@ai 반도체 시장 최근 동향 조사해줘" (naver-search-api + research)
2. "@task 이 내용으로 시장분석 보고서를 만들어서 docs/에 저장해줘"
3. "/ppt-gen 시장분석 보고서 기반으로 경영진 발표자료 만들어줘"
```

### 회의 → 기록 → 후속 조치

```
1. "/tiro 오늘 회의 녹음 요약해줘"
2. "이 회의 결과를 decisions/에 기록해줘"
3. "@task 회의에서 나온 TODO를 칸반에 등록해줘"
```

### 고객 커뮤니케이션

```
1. "이 제안서 내용으로 견적서 만들어줘" (/offer-plan)
2. "견적서 발송 안내 메일 작성해줘" (/gmail)
3. "고객에게 견적서 보냈다고 문자 보내줘" (/send-sms)
```
