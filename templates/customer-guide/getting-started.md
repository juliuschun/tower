# Tower 시작 가이드

Tower에 오신 것을 환영합니다! 이 문서는 처음 사용하시는 분을 위한 빠른 안내입니다.

---

## 1단계: 세션 열기

사이드바에서 **Sessions** 탭 → **+** 버튼을 누르면 AI와 1:1 대화가 시작됩니다.

무엇이든 물어보세요:
- "우리 팀 원칙이 뭐야?"
- "이번 주 해야 할 일 정리해줘"
- "경쟁사 분석 자료 만들어줘"

---

## 2단계: 채널에서 AI 호출하기

팀 채널에서 AI를 부르는 두 가지 방법:

**간단한 질문** → `@ai`
```
@ai 이 보고서 요약해줘
```

**본격적인 작업** → `@task`
```
@task 시장 조사 보고서를 만들어서 docs/ 에 저장해줘
```

---

## 3단계: 프로젝트 활용하기

프로젝트는 관련 파일, 대화, 채널을 하나로 묶는 공간입니다.

1. 사이드바에서 프로젝트를 선택
2. 해당 프로젝트의 파일과 맥락 안에서 AI가 작업
3. 팀원을 초대하면 모든 것이 공유됨

---

## 4단계: 스킬 사용하기

AI에게 전문 기능을 요청할 수 있습니다:

| 하고 싶은 일 | 이렇게 말하세요 |
|-------------|----------------|
| 프레젠테이션 만들기 | "AI 트렌드 발표자료 만들어줘" 또는 `/ppt-gen` |
| 견적서 만들기 | "이 내용으로 견적서 만들어줘" 또는 `/offer-plan` |
| 뉴스 검색 | "반도체 관련 최신 뉴스 찾아줘" 또는 `/naver-search-api` |
| 문자 보내기 | "고객에게 안내 문자 보내줘" 또는 `/send-sms` |
| 메일 보내기 | "이 내용으로 메일 보내줘" 또는 `/gmail` |

전체 스킬 목록은 AI에게 **"어떤 스킬이 있어?"** 라고 물어보세요.

---

## 5단계: 파일 관리

사이드바의 **Files** 탭에서 팀 파일을 탐색할 수 있습니다.

- 프로젝트별 폴더로 정리됨
- AI가 만든 파일도 여기에 저장됨
- 클릭하면 미리보기 (PDF, 이미지, 코드 등)

---

## 유용한 팁

- **구체적으로 요청**하면 더 좋은 결과가 나옵니다
  - ❌ "보고서 써줘"
  - ✅ "3월 매출 데이터를 기반으로 분기 보고서를 작성해줘. 차트도 포함해줘."

- **시각화 요청** — AI가 자동으로 차트, 표, 다이어그램을 그립니다
  - "이 데이터를 차트로 보여줘"
  - "프로세스를 플로차트로 그려줘"

- **기억력 활용** — AI는 같은 세션 안에서 이전 대화를 기억합니다
  - 한 세션에서 관련 작업을 이어서 하면 효율적
